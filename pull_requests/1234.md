## Summary

The MSI installer previously had no user-facing wizard — it installed silently with blank/default values, making the service non-functional out of the box.

This PR adds a full multi-page setup wizard with two new custom WiX dialogs.

## Changes

### New files
- `installer/RooamConfigDlg.wxs` — dialog collecting **Location Name**, **Location ID**, and **Cloud API Key** (masked)
- `installer/POSPathsDlg.wxs` — dialog collecting **SPCWIN.EXE path** and all three **XML directory paths** (pre-filled with defaults)

### Modified files
- `installer/installer.wxs` — wires `WixUI_InstallDir` + the two new dialogs into the wizard sequence; adds `LOCATION_ID` property; passes `-location-id` to `CA_WriteConfig`
- `installer/config_writer/main.go` — adds `-location-id` flag; writes `location_id` field to `rooam_config.json`
- `.github/workflows/release.yml` — `wix build` now includes `RooamConfigDlg.wxs` and `POSPathsDlg.wxs`

## Installer wizard flow
1. Welcome → 2. Licence → 3. Install Directory → 4. **Rooam Configuration** (NEW) → 5. **POSitouch Paths** (NEW) → 6. Verify Ready → 7. Install → 8. Finish
